## 质量标准

1. **数学正确性**：术语准确（convergence/divergence/limit等），公式无误
2. **双语一致性**：英文和中文内容对应，比例合理
3. **MathJax兼容**：所有数学表达式正确包装，无原始LaTeX泄露
4. **可读性**：中文流畅，符合中文数学教材习惯
5. **完整性**：所有57个section都必须完成翻译
