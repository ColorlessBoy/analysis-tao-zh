# 产品规格说明书：数学电子教科书 — Analysis I（分析学）

---

## 1. 产品概述

### 产品名称
**Analysis I 电子教科书**（数学电子教科书）

### 产品愿景
一本可以真正阅读的数学电子教材。不是扫描件，不是PDF阅读器，而是一个为数学学习优化的沉浸式阅读环境——双语并排、公式美观、导航清晰、进度可见。

### 目标用户
- 大学数学专业本科生（荣誉课程）
- 研究生复习实分析基础
- 自学数学的工程师和科学家
- 数学教师作为授课参考

### 当前实现状态

已实现：
- ✅ 双语展示（英文+中文并排）
- ✅ 深色/浅色模式切换（☀️/🌙）
- ✅ 移动端响应式（汉堡菜单）
- ✅ 命题卡片（definition/axiom/theorem/lemma）
- ✅ PDF页码参照
- ✅ URL hash导航（#1.2, #2.1）
- ✅ MathJax公式渲染

## 2. 功能规格

### P0 功能（必须做，上线前完成）

#### 2.1 阅读进度指示器
**描述：** 显示用户当前在书中的位置（章节号 + 页码/总页码）  
**位置：** 页面顶部或侧边栏  
**行为：**
- 自动跟踪当前可见的section
- 显示格式：`第 1 章 · 1.2 节 · 第 15 页 / 312 页`
- 滚动时实时更新

**验收标准：**
- [ ] 进入页面显示正确位置
- [ ] 滚动到新section后指示器更新（<500ms）
- [ ] 移动端也显示进度

---

#### 2.2 书签与高亮
**描述：** 用户可以收藏章节、给文本加高亮  
**数据存储：** localStorage（客户端）  
**行为：**
- 点击章节标题旁的书签图标 → 保存/取消
- 选中文字 → 弹出高亮按钮（黄/绿/蓝）
- 书签列表可展开查看，点击跳转

**数据结构：**
```javascript
{
  bookmarks: ["1.1", "2.3", "3.5"],
  highlights: [
    { section: "1.2", text: "limit of a sequence", color: "yellow", selectedText: "limit of a sequence" }
  ]
}
```

**验收标准：**
- [ ] 书签保存后刷新页面仍存在
- [ ] 高亮在对应section可见
- [ ] 可导出/导入书签数据

---

#### 2.3 全文搜索
**描述：** 跨英文和中文内容搜索  
**实现方式：** 客户端搜索（data/sections.json 预加载）  
**行为：**
- `/` 键打开搜索框
- 输入关键词，实时显示匹配结果（section标题 + 匹配片段）
- 点击结果跳转并高亮匹配文字
- 支持中英文搜索

**验收标准：**
- [ ] 搜索响应 <200ms（本地数据）
- [ ] 英文和中文内容均可搜索
- [ ] 匹配片段显示上下文（前后各20字）

---

#### 2.4 可折叠目录
**描述：** 左侧边栏显示章节目录，可展开/折叠  
**行为：**
- 默认显示所有章节列表
- 点击章节可展开子节
- 当前section高亮显示
- 点击跳转

**验收标准：**
- [ ] 章节展开/折叠动画流畅
- [ ] 当前section自动滚动到视口
- [ ] 移动端目录为抽屉式

---

### P1 功能（重要，下个迭代）

#### 2.5 交叉引用
**描述：** 文中引用（如"见练习2.2.3"）可点击跳转  
**识别规则：** 匹配 `\d+\.\d+\.\d+` 格式的引用  
**行为：**
- 引用文本变为可点击链接
- 点击跳转到目标位置
- hover显示引用上下文预览

**验收标准：**
- [ ] 正文中的练习引用可点击
- [ ] 引用跳转正确
- [ ] 定义/定理引用也可点击

---

#### 2.6 术语索引
**描述：**  alphabetically sorted index of mathematical terms  
**位置：** 可从目录访问的独立页面  
**数据来源：** 从sections.json自动提取  
**行为：**
- 按字母排序显示术语
- 术语显示定义位置（section号）
- 点击跳转

**验收标准：**
- [ ] 索引覆盖所有key terms
- [ ] 搜索框可快速筛选
- [ ] 中英对照索引

---

#### 2.7 脚注（Hover预览）
**描述：** 脚注不在页面底部显示，而是hover显示  
**行为：**
- 脚注标记显示为上标¹ ² ³
- hover脚注标记 → 显示tooltip
- tooltip显示脚注内容
- 不打断阅读流

**验收标准：**
- [ ] 脚注标记可识别（上标样式）
- [ ] hover显示tooltip，移开消失
- [ ] 触摸设备长按显示

---

#### 2.8 面包屑导航
**描述：** 显示当前位置路径  
**显示格式：** `第 1 章 > 1.2 节 > 为什么要学分析？`  
**位置：** 页面顶部，在进度指示器下方  

**验收标准：**
- [ ] 面包屑显示正确路径
- [ ] 每个层级可点击跳转
- [ ] 当前section不加链接

### P2 功能（增强体验）

**描述：** 每个section显示预估阅读时间

**验收标准：**

**显示位置：** section标题旁  

**验收标准：**
- [ ] 难度星级可见
- [ ] 可筛选"只看简单章节"

---

#### 2.11 目标读者指示
**描述：** 标注section适合的读者群体  
**标签：** "本科" / "研究生" / "教师"  
**显示位置：** section元信息区域

**行为：**

- [ ] 长证明可折叠（>20行）

#### 2.13 习题列表可折叠

- [ ] 练习列表默认折叠

## 3. 技术架构

### 前端技术栈
- **渲染：** 原生HTML + CSS + JavaScript（无框架依赖）
- **数学渲染：** MathJax 3.x
- **自动化测试：** Playwright
- **数据存储：** JSON文件（sections.json）+ localStorage（用户数据）

### 数据结构
```javascript
// sections.json
{
  "metadata": {
    "title_en": "Analysis I",
    "title_zh": "分析学",
    "total_pages": 312,
    "chapters": [...]
  },
  "sections": [
    {
      "id": "1.1",
      "title_en": "Introduction",
      "title_zh": "引言",
      "content_en": "...",
      "content_zh": "...",
      "difficulty": 2,
      "reading_time_minutes": 8,
      "target_audience": ["undergraduate"],
      "exercises": [...],
      "math_expressions": [...]
    }
  ]
}
```

### 性能目标
- 首屏加载 <2s
- MathJax渲染完成 <3s
- 搜索响应 <200ms
- 内存占用 <100MB
