# Analysis I · 陶哲轩实分析（中文翻译）

> Terence Tao, *Analysis I* (Fourth Edition), Hindustan Book Agency, 2022.

## 项目说明

本书是 Terence Tao《Analysis I》第四版的中文翻译对照版。

- **双语并行**：同时显示英文原文与中文译文，可自由切换
- **深浅色模式**：支持白天/夜间阅读模式
- **PDF 页码对照**：每个 section 标注对应原版 PDF 页码，方便对照原文
- **数学公式**：使用 MathJax 渲染 LaTeX 数学公式

## 目录结构

```
├── index.html          # 主页面
├── css/style.css       # 样式（含主题变量）
├── js/main.js          # 双语切换逻辑
├── data/sections.json  # 所有章节数据（双语）
└── .github/workflows/  # GitHub Pages 部署
```

## 工作流程

每节内容经过：提取 → 翻译 → 术语校验 → 逻辑审查 → 评分通过后写入。
